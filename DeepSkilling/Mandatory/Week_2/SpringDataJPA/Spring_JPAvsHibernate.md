#### JPA

- JPA helps to map java clasess to database tables and manage it using simple APIs instead of complex SQL.
- Core Components
    - Entity → defines an table
    - EntityManager → CRUD operations
    - Persistence Context
    - Criteria API
    - JPQL (Java Persistence Query Language)
    - Persistence Unit

#### Spring Data JPA

- It reduces the boilerplate code of the JPA and provides additional features as well.
- another level of abstraction over JPA implementation provider like Hibernate

#### Hibernate ORM:

- Responsible for implementing all the interfaces in the JPA to acutla code and converting java object to a relational table values
- It implements all JPA interface
